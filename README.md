# Docker PostGIS

[![Docker Pulls][pulls-badge]][docker-hub]

A Docker imge for [PostgreSQL][psql] 9.6, based on the official
PostgreSQL [docker image][hub] but with [PostGIS][pgis] 2.3 extension
installed.
